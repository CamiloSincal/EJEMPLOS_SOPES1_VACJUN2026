# Ejemplos de Laboratorio - Sistemas Operativos 1

El siguiente repositorio cuenta con ejemplos relacionados a los temas vistos dentro del laboratorio de *Sistemas Operativos 1* de la Universidad de San Carlos de Guatemala, los cuáles buscan guiar a los estudiantes paso a paso en la parte práctica del curso.

> **NOTA:** Estos ejemplos están actualizados hasta el pensum de estudios de junio de 2026. Por la naturaleza de cambio constante del curso y la licenciatura es posible que en un futuro existan temas o tecnologías que este repositorio no abarca.

## Contenido del repositorio
> Haciendo click sobre el nombre de la clase redirigirá a los archivos internos para ver el/los ejemplos.

- **[Clase 1:](./Clase%201/)** Esta carpeta cuenta con una guía de instalación básica de Docker para Ubuntu, junto con ejemplos básicos para su implementación dentro de una máquina con el sistema operativo mencionado.

- **[Clase 2:](./Clase%202/)** Dentro de esta carpeta existen 2 subcarpetas de ejemplos diferentes. Una con una guía sobre el uso de Scripts de Bash en linux y otra relacionada a la instalación de KVM en Linux.

- **[Clase 3:](./Clase%203/)** Contiene una guía para la creación de módulos en Linux, instalación de herramientas de desarrollo en C y Python, y ejemplos de módulos que exponen métricas del sistema en `/proc`.

- **[Clase 4:](./Clase%204/)** Introducción a Golang con ejemplos de programación básica, creación de daemons en Go y un servicio systemd para ejecutarlos en segundo plano.

- **[Clase 5:](./Clase%205/)** Incluye ejemplos de monitoreo con Grafana y Valkey, y una segunda sección dedicada a Flatcar Container Linux, Ignition/Butane y aprovisionamiento de VMs.

- **[Clase 6:](./Clase%206/)** Guía de instalación y uso de Rust, con explicación de `rustup`, `cargo`, creación de proyectos y conceptos básicos del lenguaje.

- **[Clase 7:](./Clase%207/)** Contiene instrucciones para levantar Zot como registro de contenedores y configurar Docker/Flatcar para usar registros privados e insecure registries.

- **[Clase 8:](./Clase%208/)** Ejemplos de gRPC en Golang y una API REST en Go, con instrucciones para generar código desde `.proto` y ejecutar servicios HTTP.

- **[Clase 9:](./Clase%209/)** Guía de Kubernetes, que incluyen conceptos de objetos como pods, deployments, services, ConfigMaps, Secrets y ejemplos de manifiestos YAML para prácticas en k8s.

- **[Clase 10:](./Clase%2010/)** Contiene un ejemplo de carga y despliegue con Locust, Gateway API y una API REST en Rust para GKE a través de un registro de ZOT, además de un subejemplo de k3s para pruebas locales. 

- **[Clase 10:](./Clase%2011/)** Explicación e implementación de RabbitMQ en clúster de kubernetes y su implementación con código de go puro y a través de Dapr.
